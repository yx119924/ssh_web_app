# SSH 移动终端 — 功能说明文档

> 本文档介绍 SSH 移动终端的所有功能和使用方法，面向最终用户。

---

## 产品简介

SSH Terminal 是一个**轻量级的手机 SSH 终端**。你在手机上打开网页，填写服务器信息，就能像在电脑上一样操作远程服务器。

**核心特点**：
- 🚀 **即开即用**：浏览器打开就能用，无需安装 App
- 🔒 **用完即走**：不保存密码、不记录日志
- 📱 **手机优化**：自适应键盘，支持横屏
- 🏠 **PWA 支持**：可添加到手机主屏幕，像原生 App

## 使用流程

### 第一步：打开应用

在手机浏览器中打开应用地址（如 `https://your-server.com`）。

### 第二步：填写连接信息

| 字段 | 说明 | 示例 |
|------|------|------|
| 主机地址 | 你要连接的服务器的 IP 地址或域名 | `192.168.1.100` 或 `myserver.com` |
| 端口 | SSH 服务的端口号 | `22`（默认值） |
| 用户名 | 服务器上的登录账号 | `root` |
| 密码 | 对应的登录密码 | （输入时会显示为圆点） |
| 访问令牌 | 如果管理员设置了，需要在此填入 | （可选） |

### 第三步：点击"连接"

点击连接按钮后：
1. 按钮变为"连接中..."（灰色不可点击）
2. 如果成功 → 自动切换到终端界面
3. 如果失败 → 显示错误信息

### 第四步：操作终端

连接成功后，你可以像在电脑终端一样操作：

- **输入命令**：打开手机键盘，直接输入
- **回车执行**：键盘上的回车/换行键
- **Tab 补全**：键盘上的 Tab 键
- **Ctrl+C**：部分浏览器支持组合键停止当前命令
- **滚动查看**：手指在终端区域上下滑动查看历史输出

### 第五步：断开连接

连接完成后，有两种方式断开：
- 点击底部状态栏的"断开"按钮
- 直接关闭浏览器页面

## PWA 安装（添加到主屏幕）

把 SSH Terminal 添加到手机桌面，像 App 一样打开：

### iPhone (Safari)

1. 在 Safari 中打开应用网页
2. 点击底部工具栏的"分享"按钮（方框+箭头图标）
3. 往下滑动，找到并点击"添加到主屏幕"
4. 确认名称后点击"添加"
5. 桌面上会出现 SSH Terminal 图标

### Android (Chrome)

1. 在 Chrome 中打开应用网页
2. 点击右上角的三个点菜单
3. 选择"添加到主屏幕"
4. 确认后桌面出现图标

安装为 PWA 后：
- 点击图标全屏打开，没有浏览器地址栏
- 加载更快（Service Worker 缓存）

## 适用场景

| 场景 | 说明 |
|------|------|
| 🔧 紧急维护 | 在外面时突然需要重启一个服务 |
| 📊 查看状态 | 快速检查服务器负载、磁盘空间 |

## 不适用场景

| 场景 | 原因 |
|------|------|
| 📂 文件传输 | 不支持 SFTP，无法上传下载文件 |
| 🔑 密钥认证 | 首版仅支持密码登录 |
| 📑 多标签操作 | 一次只能连接一台服务器 |
| ⏱️ 长时间工作 | 手机作为终端不适合长时间编码工作 |

## 安全说明

### 我们做了这些

- ✅ 密码只在使用时存在于服务器内存中，用完即丢弃
- ✅ 不记录连接日志中的密码字段
- ✅ 不保存任何密码到磁盘或数据库
- ✅ 支持 WSS 加密传输（需配置 SSL 证书）
- ✅ 浏览器端不缓存密码

### 你需要做的

- ⚠️ 使用 HTTPS/WSS 部署（公网环境必须）
- ⚠️ 设置访问令牌防止未授权使用
- ⚠️ 不要在公共 WiFi 下连接重要服务器
- ⚠️ 使用后关闭页面（密码不会残留）

## 已知限制

1. **Ctrl 组合键**：手机键盘通常没有 Ctrl 键，某些快捷键无法使用
2. **特殊终端程序**：使用 ncurses 的全屏程序（如 htop、vim）在手机上体验不佳
3. **网络切换**：WiFi 和移动数据之间切换会导致断连
4. **后台保活**：浏览器切换到后台时，WebSocket 可能被系统暂停
